# 📊 Data Analyst in Python – DataCamp Track

This repository contains my projects, exercises, and notes from the **Data Analyst in Python** career track on DataCamp.

## 🚀 About This Track
The Data Analyst in Python track covers:

- Python fundamentals
- Data manipulation with Pandas   
- Data visualization
- Statistical analysis
- Working with real-world datasets

## 🛠 Skills Learned
- Data Cleaning
- Data Wrangling
- Exploratory Data Analysis (EDA)
- Data Visualization (Matplotlib & Seaborn)
- SQL for Data Analysis
- Statistical Thinking
## 📌 Tools & Libraries Used

- Python 3
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SQL

## 🎯 Goal
To build strong data analysis skills and prepare for real-world data analyst roles.
